# Prueba Técnica FrontEnd

Esta primera prueba será de lógica de programación y manejo de datos, en una segunda instancia se hará una segunda prueba donde se verá el manejo de css como tal.

## Formularios Necesarios:

1. **Formulario de Usuario:**
   - Campos a cargar: nombre, apellido, teléfono y fecha de nacimiento.

2. **Formulario de Carga de PDF:**
   - Campos a cargar: título y archivo PDF.

3. **Formulario de Carga de Imagen:**
   - Campos a cargar: título y la imagen.

### Renderización en Pantalla:

- Mostrar en pantalla los datos del usuario, el título y archivo PDF, y el título y la imagen.
- Tanto el PDF como la imagen deben tener vista previa y ser descargables al hacer clic en ellos.

## Carrusel de Imágenes:

- Realizar un carrusel de imágenes con javascript y css (sin el uso de librerías externas).
- El carrusel se debe mover automáticamente cada 2 segundos, pasando de la imagen actual a la siguiente.
- Al poner el mouse encima del carrusel, debe detenerse el movimiento.
- La imagen del medio siempre debe ser más grande, por ejemplo, si hay 5 imágenes, la del medio debería ser de 500px, mientras que las demás podrían estar alrededor de 150px.

### Requisitos:

- La prueba será válida si se cumple como mínimo con el formulario de PDF (con visualización en pantalla) y el carrusel.
- Los demás requisitos son deseables.

### Condiciones de Desaprobación:

- La prueba se considerará desaprobada si se completan los requisitos deseables pero no los indispensables (formulario de PDF, renderizado de PDF y el carrusel).
- La prueba se considerará desaprobada si se entrega fuera de tiempo y forma.

### Condiciones de Aprobación:

- La prueba se considerará aprobada si se completan los requisitos indispensables (formulario de PDF, renderizado de PDF y el carrusel).
- La prueba se considerará aprobada si se completan tanto los requisitos indispensables como los deseables (formulario de PDF, renderizado de PDF y el carrusel).
- La prueba se considerará aprobada si se entrega en tiempo y forma.
